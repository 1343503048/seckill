公司简介
深圳市大耳马科技有限公司（简称“DARMA”）是一家致力于体征数据服务的专业型公司，2014年由胡峻浩博士创立于美国硅谷。2015年，总部迁回中国深圳。作为全球最早将高精度光纤技术应用于生命体征监测领域的研究团队，公司利用全球领先的光纤传感技术精准监测人体生命体征及行为习惯，结合大数据管理进行健康疾病状况分析，持续为养老机构、康复中心、慢病管理机构等场所的介护智能设备提供技术解决方案。
DARMA光纤传感器
DARMA光纤传感技术利用光学检测原理，使用特殊的采样模型对人体产生的微小变化进行高精度采集和还原，并借助高级算法将各种体征特点和信息分离，用以进一步的健康分析。首先，用户使用含有DARMA传感器的产品；接着，捕捉到的生命体征数据通过传感器传到DARMA服务器；最后，服务器将数据上传到App或网页以便用户查看，从而达到健康分析、安全监控和远程监护等目的。
